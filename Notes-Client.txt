A note about ECONNREFUSED errors
In the upcoming lecture at about the 4:10 timestamp, we will be moving the 
axios request from the getInitialProps function directly to the LandingPage 
as part of an explanation. This will likely fail with a long ECONNREFUSED 
error in your Skaffold output.

Node Alpine Docker images are now likely using the v16 version of Node, so, we 
will again encounter a situation that will require a catch block.

Change this code in client/pages/index.js

const LandingPage = ({ currentUser }) => {
  console.log(currentUser);
  axios.get('/api/users/currentuser');
 
  return <h1>Landing Page</h1>;
};
to this:

const LandingPage = ({ currentUser }) => {
  console.log(currentUser);
  axios.get('/api/users/currentuser').catch((err) => {
    console.log(err.message);
  });
 
  return <h1>Landing Page</h1>;
};
