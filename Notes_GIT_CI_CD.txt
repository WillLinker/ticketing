GitHub Action to run test when we pull code.
--------------------------------------------
On GitHub select "Action" from the menu bar for my repo.
Select "Simple Workflow"
rename "blank.yml" to "test.yml" and empty the file.
     name: tests
     
     on:
       pull_request
    
    jobs:
      build:
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v2
          - run: cd auth && npm install && npm run test:ci 

Notes script "test:ci" will run test one time and exit.

git status
git add . 
git commit -m "add test script"
git pull origin master
git push origin master
